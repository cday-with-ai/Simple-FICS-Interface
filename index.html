Simple FICS Interface

Welcome to Simple FICS Interface - a modern chess client for the Free Internet Chess Server.

Important: This app requires special browser headers for Stockfish analysis to work properly.
Open App (Without Headers)

⚠️ Stockfish analysis will not work without proper CORS headers

For Full Functionality:

Run the development server with proper headers:

cd packages/web
npm run start

Then open: http://localhost:5173

This launcher page exists for integration compatibility.

For the best experience, use the development server or deploy with proper CORS headers.
